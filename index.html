Everything you need

●

Live LED Indicators

Coloured LEDs in your menu bar blink green for reads, red for writes, and blend for simultaneous activity. Adjustable size, spacing, and sensitivity.

■

Disk Speed Test

Measure sequential read and write speeds across all connected drives using direct I/O that bypasses the file cache for accurate results.

▲

I/O Stress Test

Fire random-sized reads and writes at your drives simultaneously. Configurable duration, min/max I/O sizes, and per-drive statistics.

⚙

Per-Drive Colours

Customise read and write colours for each drive so you can tell at a glance which disk is active. Colours persist across sessions.

⚡

Adaptive Polling

Ramps up to 15Hz during activity and decays to 1Hz when idle, keeping CPU usage minimal. A sensitivity slider gives you control.

⎋

Safe Eject

Eject external drives directly from the menu. DiskLED releases its monitoring handles first to ensure a clean unmount.

Why Speed Test matters

A DiskLED user discovered their external SSD was running dramatically slower than expected. The Speed Test showed read/write speeds far below the drive's rating. The culprit? A USB-C charging cable that looked identical to a data cable but lacked the high-speed data lines.
